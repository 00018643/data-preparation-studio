# AI Usage Statement

AI tools were used in this project to:
- understand the coursework requirements
- explain Python, pandas, and Streamlit code
- suggest implementation ideas for app features
- help debug selected issues during development

What was verified manually:
- file upload was tested with CSV, Excel, and JSON files
- dataset overview, missing values, duplicates, type conversion, categorical tools, numeric tools, validation rules, visualization, and export functions were tested manually
- outputs were checked after each major change
- errors were reviewed and corrected manually
- final app behavior was confirmed by the student through personal testing

Final decisions, testing, and verification were completed manually by the student.
